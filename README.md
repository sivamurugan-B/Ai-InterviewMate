# 🚀 InterviewMate AI – Communication & Interview Trainer

An AI-powered chatbot that helps students improve **English communication**, **grammar**, and **interview skills** through real-time feedback.

---

## 🎯 Problem Statement

Many students struggle with:

* Fear of speaking English 😟
* Poor sentence formation ✍️
* Lack of confidence in interviews 💼

**InterviewMate AI** solves this by providing a safe environment to practice and improve.

---

## 💡 Solution

InterviewMate AI is an intelligent chatbot that:

* Conducts **mock interviews**
* Corrects **grammar mistakes**
* Suggests **better sentence structures**
* Improves **vocabulary**
* Boosts **confidence with friendly feedback**

---

## 🔥 Features

### 🎤 Interview Practice Mode

* Real-time mock interview questions
* Feedback on grammar, clarity, and confidence

### 💬 English Practice Mode

* Casual conversation training
* Sentence correction with explanations

### 📚 Vocabulary Builder

* Suggests better words and synonyms
* Provides meanings and examples

### 🤖 AI Feedback System

* Corrected sentence
* Improved version
* Encouraging feedback

---

## 🧱 Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js (Express)
* **AI Model:** Groq API (LLaMA3)
* **Deployment:** Render

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/sivamurugan-B/Ai-InterviewMate.git
cd Ai-InterviewMate
```

### 2️⃣ Install dependencies

```
npm install
```

### 3️⃣ Setup environment variables

Create a `.env` file:

```
GROQ_API_KEY=your_api_key_here
PORT=3000
```

👉 Get free API key from: https://console.groq.com

---

### 4️⃣ Run the application

```
npm start
```

Open:

```
http://localhost:3000
```

---

## 🚀 Deployment

This project is deployed on **Render**.

Steps:

1. Push code to GitHub
2. Connect repository to Render
3. Set environment variable `GROQ_API_KEY`
4. Deploy
